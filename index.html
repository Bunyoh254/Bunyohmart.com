# BunyohMart E-commerce Application ## Project Overview BunyohMart is a full-featured e-commerce web application built with Flask and Bootstrap 5. It provides a complete online shopping experience with product browsing, shopping cart functionality, order management, and a responsive design. ## Features ### Core E-commerce Features - **Product Catalog**: Browse products with images, descriptions, and pricing - **Categories**: Organize products into different categories (Electronics, Fashion, Home & Garden, Books, Sports) - **Search Functionality**: Search products by name and description - **Shopping Cart**: Session-based cart management with quantity updates - **Checkout Process**: Complete order placement with customer information - **Order Management**: Order confirmation and tracking - **Responsive Design**: Mobile-friendly interface using Bootstrap 5 ### Technical Features - **Flask Web Framework**: Backend powered by Flask - **SQLAlchemy ORM**: Database management with modern SQLAlchemy - **Session Management**: Cart persistence across user sessions - **Error Handling**: Custom 404 and 500 error pages - **JavaScript Enhancements**: Interactive features and user experience improvements ## Project Structure ``` bunyohmart/ ├── app.py # Flask application configuration and initialization ├── main.py # Application entry point ├── models.py # Database models (Product, Category, Order, OrderItem) ├── routes.py # Application routes and business logic ├── static/ │ ├── css/ │ │ └── style.css # Custom styling and responsive design │ └── js/ │ └── main.js # Interactive JavaScript features ├── templates/ │ ├── base.html # Base template with navigation and footer │ ├── index.html # Homepage with product grid │ ├── product_detail.html # Individual product page │ ├── cart.html # Shopping cart page │ ├── checkout.html # Checkout form │ ├── order_confirmation.html # Order success page │ ├── 404.html # Page not found error │ └── 500.html # Server error page └── pyproject.toml # Python dependencies ``` ## Database Models ### Category Model - `id`: Primary key - `name`: Category name (e.g., "Electronics") - `description`: Category description - `created_at`: Timestamp - `products`: Relationship to products ### Product Model - `id`: Primary key - `name`: Product name - `description`: Product description - `price`: Product price (float) - `stock`: Available quantity - `image_url`: Product image URL - `category_id`: Foreign key to category - `created_at`: Timestamp - `is_active`: Boolean for product visibility ### Order Model - `id`: Primary key - `order_number`: Unique order identifier (BO + date + random) - `customer_name`: Customer full name - `customer_email`: Customer email address - `customer_phone`: Customer phone number - `shipping_address`: Delivery address - `total_amount`: Order total - `status`: Order status (default: 'pending') - `created_at`: Order timestamp - `items`: Relationship to order items ### OrderItem Model - `id`: Primary key - `order_id`: Foreign key to order - `product_id`: Foreign key to product - `quantity`: Item quantity - `price`: Price at time of order (for historical accuracy) ## Key Routes ### Public Routes - `GET /` - Homepage with product listing and search - `GET /product/` - Individual product detail page - `GET /cart` - Shopping cart page - `GET /checkout` - Checkout form - `GET /order_confirmation/` - Order success page ### Cart Management - `POST /add_to_cart/` - Add product to cart - `POST /update_cart` - Update cart quantities - `GET /remove_from_cart/` - Remove item from cart ### Order Processing - `POST /place_order` - Process checkout and create order ### Error Handling - `404` - Page not found - `500` - Server error ## Frontend Technologies ### Bootstrap 5 - Responsive grid system - Form components - Navigation components - Card layouts - Modal dialogs ### Font Awesome Icons - Shopping cart icons - Product category icons - User interface icons ### Custom CSS Features - CSS variables for theming - Smooth transitions and animations - Product card hover effects - Custom scrollbar styling - Print-friendly styles ### JavaScript Functionality - Real-time cart updates - Form validation - Search suggestions - Quantity selectors - Loading states - Toast notifications - Smooth scrolling - Image lazy loading ## Installation and Setup ### Prerequisites - Python 3.11+ - Flask - SQLAlchemy - Gunicorn (for production) ### Dependencies ``` flask flask-sqlalchemy gunicorn psycopg2-binary email-validator ``` ### Environment Variables - `SESSION_SECRET`: Secret key for session management - `DATABASE_URL`: Database connection string (defaults to SQLite) ### Running the Application ```bash # Development python main.py # Production with Gunicorn gunicorn --bind 0.0.0.0:5000 --reuse-port --reload main:app ``` ## Configuration ### Database Configuration - Development: SQLite (`bunyohmart.db`) - Production: PostgreSQL (via `DATABASE_URL` environment variable) - Connection pooling with automatic reconnection - Health checks with `pool_pre_ping` ### Session Configuration - Filesystem-based session storage - Cart data persists across browser sessions - Session timeout configuration ## Sample Data The application initializes with sample data including: ### Categories 1.
